## Penjelasan
Terdapat 3 class yaitu Product, Hardware, dan Memory.
Kelas Product merupakan Parent dari kelas Hardware
Kelas Hardware merupakan Parent dari kelas Memory

## Atribut masing-masing kelas
1. Product
    - price
    - idProduct
2. Hardware
    - brand
    - model
3. Memory
    - frequency
    - memorySize
    - supportsCuda
